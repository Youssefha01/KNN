# Projet d'Analyse de la Santé Cardiaque

## Aperçu du Projet
Ce projet a pour objectif d'analyser des données sur la santé cardiaque afin de prédire la probabilité d'accidents cardiaques en fonction de divers indicateurs de santé. Le jeu de données comprend des informations sur la démographie des individus, leurs antécédents médicaux et leurs choix de mode de vie.

## Jeu de Données
Le jeu de données utilisé pour ce projet est `Heart_health.csv`, qui contient les colonnes suivantes :

- **ID** : Identifiant unique pour chaque individu
- **Nom** : Nom de l'individu (supprimé dans l'analyse finale)
- **Âge** : Âge de l'individu (en années)
- **Genre** : Genre de l'individu (Homme/Femme)
- **Taille(cm)** : Taille de l'individu (en centimètres)
- **Poids(kg)** : Poids de l'individu (en kilogrammes)
- **Pression Artérielle(mmHg)** : Lectures de la pression artérielle (systolique/diastolique)
- **Cholestérol(mg/dL)** : Niveaux de cholestérol (en mg/dL)
- **Glucose(mg/dL)** : Niveaux de glucose (en mg/dL)
- **Fumeur** : Statut de fumeur (Oui/Non)
- **Exercice(heures/semaine)** : Heures d'exercice moyen par semaine
- **Accident Cardiaque** : Indicateur si l'individu a eu un accident cardiaque (0 = Non, 1 = Oui)
